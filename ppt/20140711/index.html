同步 or 异步

  • 开发模式
  • 维护成本
  • 体验

同步开发流程

  • 前端写 demo
  • 开发套页面, vm. 数据
  • 联调出 bug
    • 改 demo
    • 改 vm

同步开发中的问题!

  • demo 要经过开发修改
  • 前后端分离不彻底
  • 前端没有独立开发环境
  • java 开发环境前端搭建成本高

异步开发模式

  • 前端写 html
  • 后端提供数据
  • 联调数据接口
  • 联调出 bug
    • 后端盖接口
    • 前端改 html

异步开发优点

  • 前后端分离
  • 前端有独立开发环境
  • 响应式
  • 图片retina
  • 多终端

异步实现方案

  • html 前移, 包括 layout
  • assets 前端加载
  • ui 组件化
  • 数据接口约定

app + mods

性能优化

  • 模板模块化, 预编译
  • 渲染优化
    • 页面渲染分段
    • 组件内部分次渲染

多终端实践

  • 后端修改数据接口
  • 前端替换 ui 组件
